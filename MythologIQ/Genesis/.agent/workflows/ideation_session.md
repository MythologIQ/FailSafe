# Workflow: Ideation Session (SOP-005)

This workflow is the "Phase 1: Analysis" engine. It guides the Agent to act as a **Product Manager** and **Creative Director**.

## Steps

1.  **Initiate Protocol**:

    - [ ] Create a new file: `Design Documents/[ProjectName]_BLUEPRINT.md` from `_TEMPLATE_BLUEPRINT.md`.
    - [ ] Inform User: "I am ready to capture the Vision. Let's start with the Strategic Core."

2.  **The Strategic Interview (DeepWisdom)**:

    - [ ] **Problem**: Ask "What specific pain point are we solving?"
    - [ ] **Value**: Ask "What is the ultimate value/ROI?"
    - [ ] **Competition**: Ask "Who else does this, and how are we better?"
    - [ ] **KPIs**: Ask "How will we measure success?"

3.  **The Identity Interview (TACHES)**:

    - [ ] **Target/Trouble**: Ask "Who is the audience and what makes them anxious?"
    - [ ] **Atmosphere**: Ask "What is the vibe/tone?"
    - [ ] **Content/Hierarchy**: Ask "What are the key messages and flow?"
    - [ ] **Experience/Style**: Ask "How should it feel and look?"

4.  **Synthesis & Validation**:

    - [ ] Fill out the Blueprint with the answers.
    - [ ] Present the Blueprint to the User for "Vision Lock".
    - [ ] **Goal**: Ensure the "Why" is absolute before moving to Phase 2 (Specs).

5.  **Transition**:
    - [ ] Once Blueprint is approved, suggest moving to `start_feature` workflow (Phase 2).
