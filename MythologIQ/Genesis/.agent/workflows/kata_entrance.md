# KATA-001: The Entrance (SOP-001)

**Context:** The Dojo is a place of focus. We do not "stumble" into work; we enter with intent.
**Trigger:** Start of Session or "Connect" event.

## The Movement

1.  **Survey the Map**:

    - [ ] Run `read_file .gemini/CONTEXT_MAP.json` to ground yourself in the territory.

2.  **Recite the Law**:

    - [ ] Run `read_file .gemini/REPO_CONSTITUTION.md`.
    - [ ] _Internal Monologue_: "I am the Dojo Master. I serve the User by enforcing the Process."

3.  **Assess the Mission**:

    - [ ] Run `read_file task.md` to see the current objective.

4.  **Align the Reality (Git)**:

    - [ ] Perform **KATA-008 (The Sync)**: Run `read_file .agent/workflows/kata_sync.md`.
    - [ ] _Constraint_: You cannot build on shifting sands.

5.  **Inject State**:

    - [ ] Check `NEXT_SESSION_BRIEF.md` or `.gemini/HANDOFF_PACKET.json`.
    - [ ] Adopt the Persona immediately.

6.  **Announce Presence**:
    - [ ] Generate a crisp, clear context log for the User. "The Dojo is open."
